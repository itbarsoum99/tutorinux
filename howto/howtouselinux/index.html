How do we use Linux?

Linux is used in lots of places where Windows or MacOS will not do the job, especially in the Internet of Things (smart refrigerators, Roku devices, Google Home, etc.). Linux is everywhere, because it is extremely versatile.

Day-to-day use

Linux is an operating system that can be used just like Windows and MacOS. It's easy to install, can run various applications like Windows and MacOS, and, best of all, is entirely free. Linux can be used for work, school, and life, giving you organizational advantages and a customizable interface.

Development

Linux, with a built-in bash terminal, is easily the best operating system for all things development. Giving you the ability to use tools like Visual Studio Code as well as the classic terminal allows a developer to efficiently use their skills to the fullest. It also encourages development of skills like the use of the Vim text editor that will make you a better programmer overall.

Behind the Scenes

Even if you aren't running a Linux operating system on your computer, you are still using it in some capacity. Do you have an Android phone or tablet? Android is a Linux operating system designed specifically for mobile devices. If you use Facebook, Google, Pinterest, or Wikipedia, you use Linux indirectly, too: all of those popular sites run on Linux servers. The automotive industry, including companies like Toyota, Honda, Nissan, Ford, and Mazda, use Linux for all areas of IoT in the car. The Nest Thermostat also runs Linux for its artificial learning model.
