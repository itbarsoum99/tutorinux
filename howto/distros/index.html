Linux Distributions

There are more than 25 popular different Linux distributions (and hundreds more for specific interfaces), and everyone has a favorite. This is your guide to some of the major distros.

Ubuntu

Ubuntu is one of the most-used Linux distributions, and many other distributions are based on it.

Pros:

  • Ubuntu has a very user-friendly desktop interface.
  • Ubuntu has many apps available, including most major web browsers, team communication, music, and programming applications.
  • Ubuntu comes with LibreOffice, a free, open-source office suite that's compatible with Microsoft Office, with none of the price.
  • Ubuntu comes with a built in email client as well as Mozilla Firefox as the default web browser (other browsers can be installed and made the default).
  • Regular updates and upgrades.

Cons:

  • Ubuntu has begun bringing ads to the Linux desktop, a norm-defying move.
  • Heavier distribution.
  • Heavily funded by Amazon, which has a lot of user information.
  • No Microsoft or Adobe products supported (LibreOffice is the best alternative).

Linux Mint

Pros:

  • It works out of the box, with full multimedia support and is extremely easy to use.
  • It's both free of cost and open source.
  • It's community-driven. Users are encouraged to send feedback to the project so that their ideas can be used to improve Linux Mint.
  • Based on Debian and ,Ubuntu, it provides about 30,000 packages and one of the best software managers.
  • It's safe and reliable. Linux Mint requires very little maintenance.

Cons:

  • Not as much software availability.
  • Similar desktop interface to Windows 7.
  • Outdated software centers.
  • Produces nothing that can't be found in other Debian-based distros (that may have a better interface).
  • Tries to be like Windows when they could create a better, more engaging interface.

OpenSUSE

OpenSUSE is the operating system I use on a daily basis. It has the best configuration software (YaST) and is easy to use.

Pros:

  • Truly community driven and gives its users what they want.
  • Extremely robust system that can withstand tinkering.
  • Lots of easy-to-install software.
  • YaST: the best configuration software of any operating system. Gives you complete control over every possible setting and configuration.
  • Great out-of-the-box experience: while other distributions require significant customization and work to run well, OpenSUSE doesn't.

Cons:

  • Does not come with all of the codecs for audio and video, making it difficult to play those files locally.
  • Heavier distribution, which can make it slow on older systems.

Debian

One of the earliest operating systems, it is a community-driven OS. It is also the parent of Ubuntu, which is the parent of a lot of other distros.

Pros:

  • Comes with over 59,000 packages to make your life easier.
  • Most dependable Linux operating system available, thanks to strict package policy.
  • Makes it easy to choose the level of freedom and proprietary software, as well as how stable/cutting edge you want your operating system to be.
  • Is a favored distribution of software developers.

Cons:

  • Comes with only free software, so virtually no mainstream software is included.
  • Uses Systemd, an administration tool that many Linux users see as too powerful.
  • The software is often not up to date.
  • It is a pain to install.
