Installing Linux

If you're convinced that it's time to install Linux, here's a quick tutorial. There are many distributions to choose from, but if you've made a decision, follow this tutorial!

Download the Linux File

Before we install Linux, we first have to get the file. Go to your preferred distribution's website and find the download page. Always download the largest file available (should be multiple gigabytes, not only a few hundred megabytes). This file should come in the ISO format. Once it's downloaded (usually takes 30m-1hr, depending on distribution and internet connection), move on to the next step.

Get a flashdrive or high-storage CD

Linux requires a flashdrive or other disc for installation. We recommend this SanDisk USB 3.0, which costs $6.49 on Amazon and will greatly improve installation speed. If you already have a flashdrive that you want to use, make sure that it has more storage space than the size of the Linux file, then format it(erase the contents and get it ready for use) using this tutorial.

Get Balena Etcher

Balena Etcher is the best program for writing the Linux ISO file to a flashdrive or other disc. The program quickly "flashes" the file to the disc, allowing you to begin the actual installation process.

Note: DO NOT use Rufus, UNetBootin, or other pre-installed programs on your computer. These programs are prone to error that can screw up the "flashing" process.
